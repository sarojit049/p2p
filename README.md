# PrivateConnect

**Version**: 1.0.0 (MVP)  
**Status**: Planning  
**Stack**: MERN (MongoDB, Express, React, Node.js)

PrivateConnect is a secure, invitation-only communication platform built using the MERN Stack. It is designed to allow one-to-one messaging, voice calling, and video calling without requiring mobile numbers, email addresses, OTP verification, or social logins. 

Access is strictly granted via Admin-generated Secret Codes.

## Overview

- **Secure Authentication**: Users log in using a unique Secret Code generated by the System Administrator.
- **Real-Time Communication**: One-to-one chat, voice, and video calls enabled via WebRTC and Socket.io.
- **Admin Control**: Complete user management, secret code generation, and chat monitoring via an Admin Dashboard.
